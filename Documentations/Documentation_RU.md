## `read_int`

### Описание
Читает целое число с клавиатуры. Число размером от 0 до 65535. Ввод заканчивается при нажатии клавиши Enter (Carriage Return). Введенные символы должны быть цифрами от '0' до '9'. Все введенные цифры складываются в итоговое число, которое сохраняется в указанной переменной.

### Аргументы
| Параметр | Описание |
|----------|----------|
| **%1**   | Адрес переменной, в которую будет записано введенное целое число. |

### Пример использования
```assembly
section .data
   user_input dw 0

section .text
   read_int user_input
```

## `str_to_int`

### Описание
Конвертирует строку, представляющую целое число, в соответствующее значение типа `int`. Строка должна быть завершающим символом `$` и содержать только цифры от '0' до '9'.

### Аргументы
| Параметр | Описание |
|----------|----------|
| **%1**   | Адрес строки, которую необходимо конвертировать. |
| **%2**   | Адрес переменной, в которой будет сохранено целое число. |

### Пример использования
```assembly
section .data
   num_str db "123$", 0
   result dw 0

section .text
   str_to_int num_str, result
```

## `int_to_str`

### Описание
Конвертирует целое число в строку ASCII. Число от 0 до 65535. Число может быть передано в одном из регистров (`AX`, `BX`, `DX`) или по адресу в памяти. Результат сохраняется в строковом формате в указанном буфере, где строка завершается символом `$`.

### Аргументы
| Параметр | Описание |
|----------|----------|
| **%1**   | Адрес буфера, в который будет записана строка, представляющая число. |
| **%2**   | Регистр или адрес в памяти, содержащий целое число, которое нужно преобразовать в строку. Допустимые значения: `AX`, `BX`, `DX` или адрес в памяти. |

### Пример использования
```assembly
section .data
   buffer db 10, 0 ; Буфер для строки, который должен быть достаточно длинным
   number dw 1234   ; Число, которое будет преобразовано

section .text
   int_to_str buffer, number
```

## `int_to_bin`

### Описание
Конвертирует целое число (до 5 разрядов) в двоичную строку. Результат сохраняется в строковом формате в указанном буфере. Каждый бит числа будет представлен символом '0' или '1'. Строка завершается символом конца строки `$`.

### Аргументы
| Параметр | Описание |
|----------|----------|
| **%1**   | Адрес целого числа, которое нужно конвертировать в двоичный формат. |
| **%2**   | Адрес буфера, в который будет записан результат (двоичная строка). |

### Пример использования
```assembly
section .data
   number dw 12345        ; Пример числа для конвертации
   bin_result db 17, 0    ; Буфер для хранения двоичной строки (16 бит + 1 для символа $)

section .text
   int_to_bin number, bin_result
```

## `print`

### Описание
Выводит строку на экран, используя DOS-функцию для печати строки. Строка должна быть завершается символом `$`.

### Аргументы
| Параметр | Описание |
|----------|----------|
| **%1**   | Адрес строки, которую необходимо вывести на экран. |

### Пример использования
```assembly
section .data
   message db "Hello, world!$", 0

section .text
   print message
```

## `println`

### Описание
Выводит строку на экран и добавляет перенос строки после вывода. Использует макрос `print` для вывода строки и макрос `newline` для перехода на новую строку.

### Аргументы
| Параметр | Описание |
|----------|----------|
| **%1**   | Адрес строки, которую необходимо вывести на экран с последующим переходом на новую строку. |

### Пример использования
```assembly
section .data
   message db "Hey!$", 0

section .text
   println message
```

## `print_multi`

### Описание
Выводит несколько строк на экран. Использует макрос `println` для печати каждой строки с переходом на новую строку. Поддерживает переменное количество аргументов и обязательно требует минимум два аргумента.

### Аргументы
| Параметр | Описание |
|----------|----------|
| **%1, %2, ...** | Переменное количество строк, которые необходимо вывести на экран. |

### Пример использования
```assembly
section .data
   msg1 db "Hello,", 0
   msg2 db "world!", 0
   msg3 db "How are you?", 0

section .text
   print_multi msg1, msg2, msg3
```

## `print_inline`

### Описание
Выводит несколько строк на экран в одну строку, разделяя их пробелом. Каждый аргумент выводится с пробелом между строками, за исключением последней. Поддерживает переменное количество аргументов и требует минимум два аргумента.

### Аргументы
| Параметр | Описание |
|----------|----------|
| **%1, %2, ...** | Переменное количество строк, которые необходимо вывести на экран в одной строке, разделяя пробелами. |

### Пример использования
```assembly
section .data
   msg1 db "Hello,", 0
   msg2 db "world!", 0
   msg3 db "How are you?", 0

section .text
   print_inline msg1, msg2, msg3
```

## `print_char`

### Описание
Выводит символ на экран с последующим переходом на новую строку. Сначала выводит символ новой строки (Carriage Return и Line Feed), затем выводит переданный символ.

### Аргументы
| Параметр | Описание |
|----------|----------|
| **%1**   | Символ, который необходимо вывести на экран. |

### Пример использования
```assembly
section .data
   char db 'A'  ; Символ для вывода

section .text
   print_char char
```

## `concat`

### Описание
Конкатенирует два и более строковых аргумента в одну строку. Результат записывается в первый переданный параметр, который должен быть буфером для хранения результата. Каждая строка должна быть завершена символом `$`.

### Аргументы
| Параметр | Описание |
|----------|----------|
| **%1**   | Адрес буфера, в который будет записан результат конкатенации. |
| **%2, %3, ...** | Строки для конкатенации. Каждая строка должна быть завершена символом `$`. |

### Пример использования
```assembly
section .data
   str1 db "Hello$", 0
   str2 db "World$", 0
   result db 100 dup(0) ; Буфер для результата

section .text
   concat result, str1, str2
```

## `strlen`

### Описание
Вычисляет длину строки, включая только символы до первого встреченного символа `$`. Результат сохраняется в переменную, переданную в качестве первого аргумента. Строка должна быть завершена символом `$`.

### Аргументы
| Параметр | Описание |
|----------|----------|
| **%1**   | Адрес переменной, в которой будет сохранена длина строки (целое число). |
| **%2**   | Адрес строки, для которой нужно вычислить длину. Строка должна быть завершена символом `$`. |

### Пример использования
```assembly
section .data
   my_str db "Hello$", 0
   length dw 0

section .text
   strlen length, my_str
```

## `strcopy`

### Описание
Копирует содержимое второй строки в первую строку. Строки должны быть завершены символом `$`. Вторая строка копируется в первую, включая символ конца строки `$`.

### Аргументы
| Параметр | Описание |
|----------|----------|
| **%1**   | Адрес первого буфера, в который будет скопирована строка. |
| **%2**   | Адрес второго буфера (строки), содержимое которой будет скопировано. |

### Пример использования
```assembly
section .data
   str1 db "Old string$", 0
   str2 db "New string$", 0

section .text
   strcopy str1, str2
```

## `strcmp`

### Описание
Сравнивает две строки и сохраняет результат сравнения (равны или нет) в буфер (должен быть минимум 5 байт). Строки должны быть завершены символом `$`. Если строки равны, в буфер будет скопировано `"True"`, иначе — `"False"`.

### Аргументы
| Параметр | Описание |
|----------|----------|
| **%1**   | Адрес буфера, в который будет записан результат сравнения. |
| **%2**   | Адрес первой строки, которую нужно сравнить. |
| **%3**   | Адрес второй строки, с которой нужно сравнить первую строку. |

### Пример использования
```assembly
section .data
   result db 5
   str1 db "Hello$", 0
   str2 db "Hello$", 0

section .text
   strcmp result, str1, str2
```

## `substr`

### Описание
Извлекает подстроку из строки, используя два числа, указывающих начальную и конечную позицию. Если индексы некорректны (например, конечный индекс больше длины строки или начальный индекс больше конечного), в буфер записывается пустая строка и выводится сообщение об ошибке.

### Аргументы
| Параметр | Описание |
|----------|----------|
| **%1**   | Адрес буфера, в который будет записана подстрока. |
| **%2**   | Адрес исходной строки, из которой будет извлечена подстрока. |
| **%3**   | Начальная позиция подстроки (индекс первого символа, с которого начинается извлечение). |
| **%4**   | Конечная позиция подстроки (индекс последнего символа, который включается в подстроку). |

### Пример использования
```assembly
section .data
   str db "Hello, World$", 0
   result db 10, 0

section .text
   substr result, str, 7, 12
```

</details>
